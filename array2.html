DECLARING ARRAYS


Syntax:

type arrayName [ arraySize ];


Example:

double balance[10];



INITIALISING ARRAYS


Syntax:

type arrayName[arraySize]={element1,element2,...};


Example:

double balance[5] = {1000.0, 2.0, 3.4, 7.0, 50.0};
double balance[] = {1000.0, 2.0, 3.4, 7.0, 50.0};
